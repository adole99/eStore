### Requirements

- python3

### Installation

- Clone the repo

	`git clone https://github.com/adole99/eStore.git`

- Install virtualenv

	`pip install virtualenv`

- Create Virtual environment

	`virtualenv <name>`

- Activate Virtual environment

	`source <name>/bin/activate`
